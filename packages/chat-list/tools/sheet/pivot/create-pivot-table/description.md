Intelligently recommends optimal pivot table structures and chart types based on worksheet data and user requirements to help users gain quick data insights.

Input Parameters:

- rows: Column numbers in the worksheet to be used as row labels in the pivot table
  Example: Using column 1 as row labels → rows: 1
- columns: Column numbers in the worksheet to be used as column labels in the pivot table
  Example: Using column 2 as column labels → columns: 2
- values: Column numbers in the worksheet to be used as value fields in the pivot table
  Example: Using column 3 as value field → values: 3

Key Features:

- Smart analysis of data characteristics to recommend the most suitable pivot table structure
- Automatic chart type suggestions based on data types and distribution
- Support for multi-dimensional data analysis and visualization
- Clear data insight recommendations
